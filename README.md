## What is Salesforce Platform?

Salesforce Platform is a cloud-based platform that helps companies manage customer information and business processes. It allows users to create apps, store data, automate work, and build custom solutions without needing to manage servers or hardware.

## App in Salesforce

An App in Salesforce is a collection of related tools, objects, and tabs that help users perform a specific job. For example, a Sales App helps sales teams manage customers and opportunities.

## Object in Salesforce

An Object is used to store data in Salesforce. It is similar to a table in a database. Each object contains records and fields.

Examples:
- Account
- Contact
- Opportunity
- Student

## Tab in Salesforce

A Tab is used to open and access objects or features inside Salesforce. Tabs help users easily navigate through the application.

## Difference Between Configuration and Coding

### Configuration (No Code)

Configuration means creating features using clicks and settings without writing code.

Examples:
- Creating custom fields
- Creating validation rules

### Coding (Apex)

Coding is used when advanced logic or customization is required.

Examples:
- Complex automation
- Connecting Salesforce with external systems

## How CRM Concepts Fit into Salesforce

CRM concepts like Account, Contact, and Opportunity are stored as Objects in Salesforce. These objects are organized inside Apps so users can easily manage customer information and sales processes.
